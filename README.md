# Resume on Fire!
Resume on Fire! is a resume builder followin the modern web technology trends with Next.js, Tailwind CSS and TypeScript.

### Roadmap
1. Complete the initial editor with missing resume fields with a live preview updater.
2. Persistent state management with Redux Toolkit.
3. Users and authentication with next-auth and a MongoDB Atlas cluster.
4. More resume template variations - at least 4.
